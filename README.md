## How to Open the Portfolio
1. Clone or download the repository to your local system.
2. Locate the file named `index.html`.
3. Double-click the `index.html` file or open it directly via your preferred modern web browser (e.g., Google Chrome, Brave, Safari, Firefox). No local server runtime or compilation steps required.
